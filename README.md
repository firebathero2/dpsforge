# DPS Forge

브라우저에서 실행되는 방치형 성장 게임입니다.
이 문서는 현재 코드 상태를 빠르게 파악하기 위한 최신 요약본입니다.

## 실행

1. 루트에서 정적 서버 실행
   - `python3 -m http.server 8000`
2. 브라우저 접속
   - `http://localhost:8000`

빌드 과정은 없습니다.

## 프로젝트 구조

- [index.html](index.html): 화면 마크업, 스크립트 로드
- [css/style.css](css/style.css): 전체 스타일
- [js/constants.js](js/constants.js): 밸런스 상수, 수식, 초기 상태
- [js/gameLogic.js](js/gameLogic.js): 게임 상태 변경 및 규칙 엔진
- [js/ui.js](js/ui.js): 렌더링, 이벤트, 자동화, 튜토리얼 UI

## 현재 핵심 규칙

### 기본

- 최대 단수: 40
- 판매 가능 시작 단수: 15
- 시작 골드: 0
- 1단 구매 가격: 1500
- 틱 간격: 0.1초

### 수입

- 배치된 유닛 DPS 기반으로 골드가 누적됩니다.
- 수입 배율은 DPS 로그 구간을 기준으로 증가합니다.

### 강화

- 기본 강화 확률은 [js/constants.js](js/constants.js)의 `UPGRADE_PROBABILITY`를 사용합니다.
- +1/+2/+3 강화 확률은 [js/gameLogic.js](js/gameLogic.js)에서 판정합니다.
- 강화확률 특성 보너스는 39단까지 적용됩니다.
- 35단: +3강 확률은 +1강으로 편입됩니다.
- 36단 이상: +2/+3강 확률이 모두 +1강으로 편입됩니다.

### 중간보스

- 최대 8회 클리어 보상
- 도전 시간: 3초 평균 DPS 판정
- 현재 컷(1~8회): `90, 200, 360, 650, 1200, 2100, 3800, 6800`
- 8회 완료 후 다음 컷 표시는 `-`

### 특성(요약)

- 공격력 업글: 비용 4, 최대 50
- +1강 확률: 비용 2, 최대 100
- +2강 확률: 비용 40, 최대 50
- +3강 확률: 비용 200, 최대 30
- 슬롯 증가: 비용 1, 최대 20
- 자동화 속도: 비용 2, 최대 10

정확한 수치는 [js/constants.js](js/constants.js)의 `TRAIT_SYSTEMS`가 소스 오브 트루스입니다.

### 자동화

- 자동구매/자동강화/자동판매를 지원합니다.
- 자동구매는 단수별 토글이 있으며 동작 간격은 자동화 속도 특성과 환생 보상 영향을 받습니다.

### 저장

- LocalStorage 키:
  - `dpsforge_gamestate`
  - `dpsforge_automation`
  - `dpsforge_lastsave`
- 상태 변경 감지 기반 자동 저장(최소 간격 3000ms)

## 참고

- 튜토리얼 및 디버그 패널 UI가 포함되어 있습니다.
- 최신 밸런스 조정은 상수 파일([js/constants.js](js/constants.js))을 우선 확인하세요.
